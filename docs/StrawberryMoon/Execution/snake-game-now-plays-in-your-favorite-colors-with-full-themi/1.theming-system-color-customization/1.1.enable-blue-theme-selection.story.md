# Enable Blue Theme Selection

## User Story Statement
As a player, I want to select a Blue theme from pre-built options, so that I can customize the game's appearance to match my visual preferences without losing gameplay functionality.

## Acceptance Criteria
- Blue theme (Ocean Blue) appears in the theme selector dropdown alongside other pre-built themes (Cyberpunk, Retro Green, Neon Pink, Monochrome)
- Selecting Blue theme instantly applies the color palette to snake gradient, food glow, grid overlay, background, and text elements
- Blue theme persists in browser local storage and automatically loads on next game session
- Theme switching does not interrupt active gameplay or reset game state
- Blue theme meets WCAG AA contrast ratio standards (4.5:1 for text, 3:1 for graphics)
- Theme renders with <5ms performance impact to frame time
- No additional bundle size increase beyond 15KB total for theming system
